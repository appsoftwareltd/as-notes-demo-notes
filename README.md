# AS Notes Demo Notes

This is the demo / test note set for the VS Code extension AS Notes: https://github.com/appsoftwareltd/as-notes

You can use this repository as a set of test notes if you haven't yet set up a knowledge base.

Once you have cloned this repo, you can run the VS Code commands:

```
AS Notes: Initialise Workspace
AS Notes: Rebuild Index
```

AS Notes is installable via Visual Studio Marketplace: https://marketplace.visualstudio.com/items?itemName=AppSoftwareLtd.as-notes

Report issues at https://github.com/appsoftwareltd/as-notes/issues
